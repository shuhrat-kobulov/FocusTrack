# FocusTrack

A professional Electron JS desktop application which tracks usage time of different applications (active ones) running on your computer and displays it along with detailed analysis reports.

## Usage

-   Clone/Download the source code
-   Install all requirements
    ```
    npm install
    ```
    or
    ```
    yarn install
    ```
-   Run :

    ```
    npm start
    ```

    or

    ```
    electron .
    ```

-   The application window will open

-   Navigate through the project
