About FocusTrack

FocusTrack was born from a simple need: understanding where our time goes during the workday. Built with modern web technologies and packaged as a desktop application, it provides the perfect balance of functionality and simplicity.

Originally developed as an academic project for NTCC University, FocusTrack demonstrates practical application of modern web technologies in desktop application development using Electron.
